3DViewer_v1.0

Документация по использованию

1. Общее

Данная программа предназначена для визуализирования Object файлов с расширением .obj. Визуализируются только вершины и полигоны, образуя множество точек и линий между ними - рёбер.

Имеется ряд настраиваемых параметров:

  • стиль точек
    • отсутствующий
    • шары
    • квадраты
  • радиус точек
  • цвет точек
  • стиль линий
    • сплошной
    • пунктирный
  • радиус линий
  • цвет линий
  • цвет фона
  • угол поворота по оси X
  • угол поворота по оси Y
  • перемещение по оси X
  • перемещение по оси Y
  • перемещение по оси Z

2. Открытие файлов

Программой поддерживается открытие только файлов с расширением .obj. Чтобы открыть файл, необходимо выбрать соответствующий пункт меню и кнопку "Open file".

Также при перезапуске программы предыдущий открытый файл и все его настройки восстанавливаются.

3. Перемещение модели и задание цвета её элементам

В графическом интерфейсе программы предусмотрены различные "крутилки", "слайдеры" и выпадающие меню с выбором стилей точек или линий.

Задать цвет точкам, рёбрам или фону возможно благодаря "крутилкам" в правой части окна программы. Цвет настраивается путём комбинации трёх цветов - красного, зелёного и синего.

Для перемещения модели предусмотрены сочетания клавиш и слайдеры внизу окна программы. Таким образом, можно либо потянуть модель мышью, либо двигать слайдеры.

Для непосредственно перемещения модели есть клавиши W, S, A, D, R, F. Клавиша Shift увеличивает скорость перемещения модели.

4. Создание gif-анимаций

Для создания .gif-изображения нужно выбрать пункт меню Save File и выбрать пункт подменю .gif. После этого Вам будет предложено выбрать путь к сохраняемому файлу и начнётся запись .gif-изображения. Об этом будет говорить надпись справа сверху Recording image. Запись ведётся с момента выбора пути файла и его имени 5 секунд.

5. Сохранение изображений

Для сохранения изображения текущего состояние 3d-модели нужно выбрать пункт меню Save File. Сохранить можно в двух форматах: .bmp и .jpeg. После выбора нужного пункта меню откроется диалоговое окно с выбором пути и имени файла.
